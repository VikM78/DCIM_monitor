# DCIM Monitoring System

Комплексная система мониторинга аварий DCIM с email-уведомлениями.

## 📦 Модули проекта

### 1. monitor_and_send_email (основной, работает постоянно)
- Мониторинг изменений в таблице `events_recs`
- Отслеживание update_type (1-появление, 2-исчезновение, 3-подтверждение)
- Отправка email-уведомлений по расписанию
- Логирование всех событий

### 2. email_manager (веб-интерфейс)
- Управление списком email-получателей
- Просмотр статистики
- API endpoints
- **Подписки на события планировщика** (v2.1.0)

### 3. schedule_email (планировщик)
- Настройка расписания отправки
- Выбор дней недели и времени
- Выбор типов событий для отправки
- **События без привязки ко времени** (v0.005)

## 🚀 Установка

```bash
# Клонирование репозитория
git clone <repo-url>
cd DCIM_project

# Установка зависимостей
pip install -r requirements.txt

# Настройка окружения
cp .env.example .env
# Отредактируйте .env файл

# Запуск мониторинга (основной сервис)
python run.py -m

# Запуск веб-интерфейса (опционально)
python run.py -e

# Запуск планировщика (опционально)
python run.py -s
