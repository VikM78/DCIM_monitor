# Changelog - Email Manager

Все значимые изменения будут отражены в этом файле.

## [2.1.0] - 2026-06-19

### Added
- ✨ Система подписок на события планировщика
- ✨ Страница управления подписками для каждого email
- ✨ Возможность выбирать расписания для получения уведомлений
- ✨ Отображение типов событий в интерфейсе подписок

### Changed
- 🔄 Обновлён интерфейс списка email (добавлена кнопка "Подписки")
- 🔄 Улучшена навигация между модулями

## [2.0.0] - 2026-06-15

### Added
- ✨ Production версия с поддержкой .env
- ✨ Health check endpoint (/health)
- ✨ API endpoint для получения списка email (/api/emails)
- ✨ Статистика по email (/stats)
- ✨ Информация о версии (/version)
- ✨ Логирование с ротацией файлов
- ✨ Валидация email на сервере
- ✨ Индексы в базе данных
- ✨ Поля created_at и updated_at
- ✨ Триггер для автоматического обновления updated_at
- ✨ Защита cookies (Secure, HttpOnly, SameSite)

### Changed
- 🔄 Префикс переменных БД с DB_ на DB_MESSAGE_
- 🔄 Переход на переменные окружения
- 🔄 Улучшенная обработка ошибок
- 🔄 Повторные попытки подключения к БД (3 попытки)
- 🔄 Email приводится к нижнему регистру

### Fixed
- 🐛 Ошибка дублирования email
- 🐛 Проблемы с кодировкой
- 🐛 Утечка информации в логах

### Security
- 🔒 Генерация секретных ключей через secrets
- 🔒 Защита от CSRF
- 🔒 Безопасные cookies
